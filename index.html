Image Recognition

This project involves handwriting recognition using the Support Vector Machine (SVM) algorithm on the digits dataset from scikit-learn. The dataset consists of images of handwritten digits. The SVM model is trained on a subset of the data, and its accuracy is evaluated. The code includes visualization of selected images, prediction of their labels, and assessment of the model's overall accuracy. The project aims to demonstrate the application of machine learning in recognizing handwritten digits.

Google gemini LLM

This project utilizes Streamlit to create a simple Question and Answer (Q&A) demo application powered by the GEMINI language model from Google's GenerativeAI. It loads environment variables using `python-dotenv`, and the GEMINI API key is stored in a secure manner. The application takes user input through a text box, and upon clicking the "Ask the question" button, it leverages the GEMINI model to generate responses. The output is then displayed using Streamlit. The project showcases a straightforward integration of a generative language model for Q&A applications.
